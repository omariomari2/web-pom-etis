Installation Guide

Get started with PomoDoro-Play in just a few simple steps using Chrome Developer Tools

1

Download the Source Code

Clone or download the repository from GitHub

git clone https://github.com/omariomari2/Pomodoro-Play.git
2

Open Chrome Extensions

Navigate to the extensions page

chrome://extensions/
3

Enable Developer Mode

Toggle the switch in the top-right corner

4

Load Unpacked

Click 'Load unpacked' and select the extension directory

Need help? Check out the GitHub repository for more details.
